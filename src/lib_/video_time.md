# `video_time.py` の切り出し指定仕様

[video_time.py](video_time.py)

`video_time.py` は、動画処理コマンドで使う `--start`、`--end`、`--duration` の指定を解釈する共通処理である。

## 指定できる値

`--start` と `--end` には、時刻、フレーム、または `last` を指定できる。

- 時刻: `10`, `1.5`, `1:30`, `0:02.400`
- フレーム: `frame:20`, `f:20`, `f20`
- 末尾: `last`

`--duration` には、時刻またはフレーム数を指定できる。

- 時刻: `5`, `0:05`, `2.5`
- フレーム数: `frame:30`, `f:30`, `f30`

`--duration=last` は不可。

## 基本ルール

`--start`、`--end`、`--duration` は最大 2 つまで同時に指定できる。3 つすべてを指定するとエラーになる。

秒単位の切り出しでは、最終的に `start` と `duration` に変換される。フレーム単位の切り出しでは、最終的に開始フレームと終了フレームに変換される。

`--end` は「その位置の直前まで」を意味する。たとえば `--start=f10 --end=f20` の範囲は、フレーム 10 から 19 までになる。

## 組み合わせごとの挙動

### 何も指定しない

フレーム範囲では動画全体になる。

秒単位の切り出しでは、呼び出し側が「最低 1 つ必要」としている場合はエラーになる。許可されている場合は、開始位置も長さも未指定になる。

### `--start` のみ

指定位置から動画末尾までを対象にする。

フレーム単位では、`--start` が動画末尾以降になるとエラーになる。

### `--end` のみ

動画先頭から `--end` の直前までを対象にする。

`--end` は 0 より大きい必要がある。

### `--duration` のみ

動画先頭から指定した長さ分を対象にする。

`--duration` は 0 より大きい必要がある。

### `--start` と `--end`

`--start` から `--end` の直前までを対象にする。

`--end` は `--start` より後でなければならない。

### `--start` と `--duration`

`--start` から指定した長さ分を対象にする。

`--duration` は 0 より大きい必要がある。

### `--end` と `--duration`

`--end` の直前で終わるように、指定した長さ分を逆算して対象範囲を決める。

`--duration` は 0 より大きい必要がある。`--end - --duration` が動画先頭より前になる場合はエラーになる。

## `last` の意味

`last` は、指定する場所によって意味が少し変わる。

- 単一フレーム参照では、最後の実フレームを表す。
- 切り出しの `--end` では、動画末尾の境界を表す。
- 秒単位に変換する場合は、動画の duration が必要になる。

そのため、`--end=last` は動画末尾までの切り出しに使える。一方で、`--duration=last` は使えない。

## フレームへの変換

時刻をフレームに変換する場合は、`round(秒数 * fps)` で丸める。

単一フレーム参照は動画内の有効な範囲に丸められる。切り出し境界は `0` から `frame_count` までに丸められる。
